# class-placement — 분반 배치 시스템

학생을 여러 반으로 나눠 배치(분반)하는 웹앱. 엑셀 명단을 업로드하면 성적·성별 등 기준에 따라 균등 배치하고, 드래그앤드롭으로 수동 조정 및 교체 추천을 제공한다. React + SheetJS 기반 단일 HTML 파일.

## 메인 파일

| 파일 | 용도 | 비고 |
|------|------|------|
| `분반배치_v2.html` | **개발본 / 편집용 원본** | React·Babel·SheetJS를 CDN에서 로드 (인터넷 필요) |
| `분반배치_오프라인_v2.html` | **실행본 / 배포용** | 라이브러리를 파일에 내장 — 인터넷 없이 학교 PC에서 바로 실행 (약 4MB) |

두 파일은 같은 v2 버전의 "개발본 / 오프라인 배포본" 관계다. 코드를 고칠 땐 `분반배치_v2.html`을, 실제 사용은 `분반배치_오프라인_v2.html`을 쓴다.

## 주요 기능

- 엑셀(.xlsx) 학생 명단 업로드
- 성적·성별 기준 균등 분반 자동 배치
- 반별 평균 분포 시각화
- 드래그앤드롭 수동 이동
- 학생 교체 추천 (배치 균형 유지)
- 결과 엑셀 내보내기

## `_archive/`

개발 과정의 구버전·연습본과 데이터 파일 보관소.

- 구버전 HTML: `분반배치_v1`, `[범용ver1]`, `[범용추가]` 등 — v2 이전 단계
- **학생 실명 명단 데이터(.xlsx / .numbers / .gsheet)와 명단이 하드코딩된 `[하드코딩]*.html` 은 `.gitignore`로 커밋에서 제외**되어 로컬에만 존재한다 (개인정보 보호).

## 이력

바이브코딩 입문 시기에 채팅으로 제작. 진행 순서: `범용` → `범용 ver.1` → `v1` → `v2` → `오프라인 v2`.
