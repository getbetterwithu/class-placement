▶ 프로그램 실행하기

? 이 프로그램은 무엇인가요?

새 학년 분반(반 편성)을 성적 균형 + 분리 대상 + 직접 조정까지 한 화면에서 처리하는 도구입니다. 엑셀(또는 CSV) 한 장을 올리면, 성적을 고르게 나눈 ㄹ자(지그재그) 배치를 자동으로 만들어 줍니다.

🔒 개인정보 안심 — 모든 처리는 여러분의 브라우저 안에서만 일어납니다. 서버로 전송되지 않으며, 작업 이력도 남지 않습니다. (자세한 내용은 아래 7번)

1 시작하기

  1. 학년 선택 — 첫 화면에서 1학년 → 2학년 또는 2학년 → 3학년을 고릅니다. (현재 학년이 아니라 “어느 학년을 분반하는가”를 기준으로 고르면 됩니다.)
  2. 학생 데이터 업로드 — 초록색 학생 데이터 업로드 버튼을 누르거나, 그 영역에 파일을 끌어다 놓으면(드래그 앤 드롭) 됩니다.
  3. ㄹ자 배치 실행 — 상단 🎯 ㄹ자 배치 실행을 누르면 자동 배치가 완성됩니다.

2 학생 데이터 형식

엑셀 첫 줄에 아래 열 이름(헤더)이 있어야 합니다. 열 순서는 상관없습니다.

열 이름필수설명예시
구반✅이전 학년의 반1
구번호✅이전 학년의 번호4
성명✅학생 이름일영사
기준성적✅배치 기준이 되는 점수1077.6
성별선택남/여 (있으면 화면에 표시)남
생년월일선택비워 두어도 됨(공란)
💡 반 개수는 자동 감지됩니다. 구반 값의 최댓값을 보고 8반이든 10반이든 학교에 맞게 처리합니다. 남녀공학이면 성별에 남·여를 섞어 두면 각 반의 성비가 함께 표시됩니다.
⚠️ 구반·구번호는 이전 학년(배치 전) 정보를 넣습니다. 배치 결과로 정해질 새 학번이 아니라, 지금까지 쓰던 반·번호입니다.

3 ㄹ자(지그재그) 배치란?

전교생을 기준성적 순으로 줄 세운 뒤, 1반→2반→…→마지막반→마지막반→…→2반→1반 순으로 뱀이 기어가듯(ㄹ자) 한 명씩 나눠 담는 방식입니다. 이렇게 하면 모든 반의 평균 성적이 고르게 맞춰집니다.

기준성적이 비어 있거나 0인 학생은 맨 마지막에 이어서 배치됩니다.

4 분리해야 할 학생 설정

같은 반이 되면 안 되는 학생 쌍을 등록하면, 한 반에 같이 묶였을 때 빨갛게 경고 표시됩니다.

  • 엑셀 업로드 — A열·B열에 분리할 두 학생을 적은 파일을 올립니다. (이름 또는 학번)
  • 수동 추가 — + 수동 추가 버튼으로 직접 한 쌍씩 입력할 수 있습니다.
  • 접기/펼치기 — 목록 제목(🚫)을 클릭하면 명단을 감췄다 보였다 할 수 있습니다.
한 학생이 여러 명과 분리 대상이면, 쌍을 따로따로 나눠 적습니다. (김 vs 이·박 → “김 이”, “김 박” 두 줄)

5 직접 조정하기

  • 드래그로 교체 — 학생 카드를 다른 반 학생 위로 끌어다 놓으면 두 학생의 반이 맞바뀝니다.
  • 되돌리기 / 다시 실행 — 상단 ↩️ 되돌리기(Ctrl+Z), ↪️ 다시 실행(Ctrl+Y).
  • 학급별 분산 확인 — 이전 학년 반 버튼을 누르면, 그 반 학생들이 새 반에 어떻게 흩어졌는지 강조됩니다.
  • 원래 배치로 — 손댄 것을 처음 ㄹ자 배치 상태로 한 번에 되돌립니다.

6 저장 · 내보내기

버튼하는 일
💾 작업 저장현재 상태를 JSON 파일로 내려받습니다. 나중에 이어서 작업할 때 사용합니다.
📊 엑셀 다운로드최종 반 명렬표 + 신학번 명단을 엑셀로 내려받습니다.
작업 파일 불러오기첫 화면에서 저장해 둔 JSON 파일을 올려 작업을 복원합니다.

7 개인정보 · 작업 이력

🛡️ 이 웹 버전은 작업 이력을 브라우저에 저장하지 않습니다. 새로고침하거나 창을 닫으면 작업이 사라집니다. 여러 학교가 같은 PC로 접속해도 앞사람의 명단이 남지 않습니다.

따라서 작업을 보관하려면 반드시 💾 작업 저장(JSON) 또는 📊 엑셀 다운로드로 파일을 직접 받아 두세요. 받은 파일은 학교 PC나 USB에 안전하게 보관합니다.

8 샘플 파일로 먼저 체험하기

실제 명단이 없어도 아래 익명화 샘플로 모든 기능을 시험해 볼 수 있습니다. (이름은 일영일·이영삼처럼 가명으로 바꾼 데이터라 개인정보가 없습니다.)
